# TexECA Arduino Training

Beginner Arduino training module developed for the TexECA Internal Training Workshop. This workshop introduces participants to the fundamentals of Arduino programming, circuit development, sensors, actuators, and simple embedded system applications through hands-on activities and mini projects.

---

## 📚 Topics Covered

### Module 1 — Blink & Serial Monitor
- Arduino program structure
- Digital output
- Serial communication
- Basic debugging

### Module 2 — Button
- Digital input
- INPUT_PULLUP
- Conditional logic
- Button latch system

### Module 3 — Potentiometer & Buzzer
- Analog input
- PWM output
- Value mapping
- Sensor-to-output control

### Mini Project — Traffic Light System
- State sequencing
- Timing control
- Multi-output system

### Module 4 — Ultrasonic & Servo
- Distance measurement
- Ultrasonic sensor
- Servo motor control

### Mini Project — Autogate System
- Automation logic
- Object detection
- Real-world embedded system simulation

## 🚀 Getting Started

1. Install Arduino IDE
2. Connect Arduino Uno to your computer
3. Open the desired `.ino` file
4. Select:
   - Board → Arduino Uno
   - Correct COM Port
5. Upload the code to the Arduino board

---
## ⚠ Troubleshooting

Common issues:
- Wrong COM port selected
- Loose wiring connection
- Incorrect LED polarity
- Baud rate mismatch in Serial Monitor

Always double-check:
- Wiring
- Pin numbers
- Power connection
- Serial baud rate
